# Employee Management System

A production-style Employee Management System built using Spring Boot.

## Tech Stack

- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 Database (Development)
- MySQL (Production)
- Maven
- REST APIs

## Features

- Role Management
- Employee Management
- Department Management
- User Authentication (JWT)
- Exception Handling
- Validation
- Swagger API Documentation

## Project Structure

```text
Controller
   ↓
Service
   ↓
Repository
   ↓
Database
```
